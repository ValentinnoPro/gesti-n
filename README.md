Sistema de Gestión de Stock y Ventas - "Kiosco Familiar" :
Este proyecto surge como una solución práctica para la digitalización y optimización de un comercio familiar. Desarrollado con Python y Django, el sistema permite gestionar de manera eficiente el inventario, los registros de clientes y el flujo de ventas.

Características principales:
Gestion de Productos: Registro, actualización de stock y categorías de mercadería.

Control de Ventas: Interfaz para el registro de transacciones en tiempo real.

Administración de Clientes: Base de datos para el seguimiento de cuentas y perfiles.

Lógica de Negocio: Aplicación de conocimientos de Economía y Administración para la resolución de problemas de conciliación de caja y stock.

 Tecnologías utilizadas:
Backend: Python y Django .

Base de Datos: SQLite (desarrollo) con diseño preparado para migración a MySQL.

Frontend: Templates dinámicos con HTML5 y BOOSTRAP5.

Impacto del Proyecto:
El objetivo principal fue reemplazar los registros manuales por una herramienta digital que permita un inventarios más precisO y rápidO, reduciendo el margen de error en la gestión diaria del kiosco.
